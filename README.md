# Tic Tac Toe Game in C++

A simple two-player Tic Tac Toe game played on the console.

## Features

- Two-player mode (Player 1: x, Player 2: o)
- Player name input
- Clean board display
- Win and draw detection
- Press `ESC` to exit after game ends

## Menu

1) Players take turns entering numbers (1-9) to place their mark
2) Board positions:
```
1 | 2 | 3
4 | 5 | 6
7 | 8 | 9
```

## How to Play

1. Run the program
2. Enter names for both players
3. Take turns entering a number (1-9) to place your mark
4. First player to get 3 in a row wins

## Limitations

- Windows only (uses `windows.h` and `conio.h`)
- No input validation for out-of-range or already-taken positions (simple restore needed)
- No restart option (game exits after finish)
